## Installation

### Prerequisites

* Python3
* Google Chrome

### Installation Steps
#### Windows
`cd Drive:\path\to\folder`

`py -m venv env`

`.\env\Scripts\activate`

`pip3 install -r .\requirements.txt`

#### Linux
`cd /path/to/folder`

`python3 -m venv env`

`source env/bin/activate`

`pip3 install -r ./requirements.txt`
